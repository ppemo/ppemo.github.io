## 关于本站 (About This Site)

欢迎来到我的个人博客，这里是我记录思考、分享知识和探索技术的小天地。本站的建立，源于对技术分享的热情和对知识沉淀的渴望。希望这里的内容能对你有所启发或帮助。

## 技术栈 (Technology Stack)

本站基于以下技术构建和运行，向所有开源贡献者致敬：

*   **静态网站生成器 (Static Site Generator)**：[Jekyll](https://jekyllrb.com/) - 一个简单、博客友好的静态网站生成器，它将纯文本转换为静态网站。
*   **博客主题 (Blog Theme)**：[Chirpy](https://github.com/cotes2020/jekyll-theme-chirpy) - 一个优雅、响应式且功能丰富的 Jekyll 主题，它为本站提供了出色的外观和用户体验。特别感谢其开发者 `cotes2020` 及其社区的贡献。
*   **托管平台 (Hosting Platform)**：[GitHub Pages](https://pages.github.com/) - 免费且高效的静态网站托管服务，借助 GitHub Actions 实现自动化部署。
*   **前端技术 (Frontend Technologies)**：HTML, CSS (SCSS), JavaScript - 用于构建页面结构、样式和交互。
*   **版本控制 (Version Control)**：[Git](https://git-scm.com/) / [GitHub](https://github.com/) - 用于管理代码版本和协作。
